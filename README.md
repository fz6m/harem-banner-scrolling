# harem-banner-scrolling

一个门户首页，可以让你的后宫们在背景无缝、无限滚动。

### 开发
```bash
    yarn
    yarn dev
```
之后访问 `127.0.0.1:9527` 就能看到页面。

### 构建
```bash
    yarn build
```
